# Hurtig element-oprettelse med auto-connector

Goer det nemmere at oprette flere elementer i traek, og tilfoej automatisk connector mellem dem.

## Opgaver

- [x] "Sticky chain mode": efter oprettelse af et element, forbliv i samme tool saa man hurtigt kan klikke igen for at oprette det naeste
- [x] Auto-connector: naar et nyt element oprettes i chain mode, tegn automatisk en connector fra det forrige element til det nye
- [x] Escape eller Select-tool afslutter chain mode
- [x] Visuel feedback (f.eks. preview-linje fra forrige element til cursor)

## Acceptkriterier

1. Vaelg Sticky Note tool → klik for at oprette → tool forbliver aktiv → klik igen for at oprette endnu en
2. Connector tegnes automatisk fra foerste sticky til anden sticky
3. Tryk Escape eller V for at stoppe og gaa tilbage til Select
