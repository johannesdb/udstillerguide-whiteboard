# Billeder: drag-and-drop og paste paa canvas

Tilfoej support for at traekke billeder fra filsystemet og paste fra clipboard direkte paa whiteboard canvas.

## Opgaver

- [x] Ny element-type `image` registreret i plugin registry (14. type)
- [x] Backend endpoint til upload af billeder (`POST /api/boards/:id/images`)
- [x] Backend endpoint til hentning af billeder (`GET /api/boards/:board_id/images/:image_id`)
- [x] Canvas drag-and-drop handler (dragover/drop events)
- [x] Canvas paste handler (Ctrl+V / Cmd+V fra clipboard)
- [x] Tegn billede-elementer paa canvas (draw funktion med loading placeholder)
- [x] Hit-test for billede-elementer (pointInRect)
- [x] Resize handles for billeder (standard selection handles)
- [x] Sync billede-elementer via WebSocket (src URL synces som del af element JSON)

## Acceptkriterier

1. Traek en billedfil fra Finder/Explorer ind paa canvas → billedet vises
2. Kopier et billede i en anden app, Ctrl+V paa canvas → billedet vises
3. Billeder kan flyttes og resizes som andre elementer
4. Billeder synkroniseres til andre brugere via WebSocket
